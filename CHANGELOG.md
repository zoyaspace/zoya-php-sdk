# Changelog

All notable changes to this package will be documented in this file.

## 0.1.1

- lowered the runtime requirement from PHP 8.3 to PHP 8.2
- aligned CI and development dependencies with PHP 8.2
- kept the SDK API surface unchanged while broadening compatibility

## 0.1.0

- introduced the first public SDK foundation for the Zoya public API
- added public API environment and version configuration
- added API key authentication with `X-Zoya-Api-Key`
- added convenience methods for public investments, buildings, units, products, lead sources, sales channels, and leads
- added an OpenAPI-first contract scaffold
- added a default client factory, examples, and package CI
